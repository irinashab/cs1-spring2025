programming assignments naming and submission guidelines

When submitting a programming assignment, please ensure that you adhere to the following requirements:

  1. File Naming Convention:

    • Each programming sketch should be named following this format: firstInitialLastNameAssignmentName#Question#.pde
      • For example: iShablinskyLab1Q2.pde or iShablinskyHW1Q1.pde.
      • Please note that the .pde extension will automatically be assigned to the name,  and you don't have to type it.
      • No spaces or special characters except the underscore are allowed in the sketch names.

  2. Top-of-File Comments:

    • Every sketch must include comments at the top that specify:

      • Your First Name
      • Your Last Name
      • Assignment Name
      • Question Number
      • Date the assignment was completed

    • For instance:

      //Alan Turing
      //homework 3
      //Question 2
  3.              // 01/23/1937
  5. To submit your work:
    • Locate your sketch folder (you can use Sketch-->Show Sketch Folder menu option)
    • Compress the sketch folder
    • Upload the compressed sketch

If you are submitting multiple sketchbooks (i.e. sketches in their folders)  for a single assignment, would you please place them in a single folder and then compress that folder before submission?
